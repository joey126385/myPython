#### ~3.1.1.2	對【字典】整體url編碼

~~~python
import urllib.parse
data = {
    "name": "$joey",
    "pwd": "123"
}
# 對data 整體編碼
url_params_str = urllib.parse.urlencode(data)

# {1} name=%24joey&pwd=123
print("{1}", url_params_str)  # name=%24joey&pwd=123

data = urllib.parse.parse_qs(url_params_str)
# {2} {'name': ['$joey'], 'pwd': ['123']} <class 'dict'>
print("{2}", data, type(data))  # {'name': ['$joey'], 'pwd': ['123']} <class 'dict'>

# {3} dict_items([('name', ['$joey']), ('pwd', ['123'])])
print("{3}", data.items())

# {4} {'name': '$joey', 'pwd': '123'}
print("{4}", {key: val[0] for key, val in data.items()})
~~~

## 3.2	摘要與加密算法



### 3.2.1	摘要算法：MD5

1. ​	不可逆.所以.摘要算法就不是一個加密邏輯
2. ​    相同的內容計算出來的摘要是一樣的，不同的內容計算出來的結果差別非常大
3. ​     結果的長度是固定的，跟算法有關係，
4. ​     一般MD5值是由32位元有數字[0~9]和字母[a~f]所組成的字符串

#### ~3.2.1.1	MD5摘要算法

~~~python
from hashlib import md5, sha1

# (1)   md5實例化
md5_obj = md5()

# (2)   update更新數據:對數據做摘要
data = "你好root123456".encode()  # b'\xe4\xbd\xa0\xe5\xa5\xbdroot123456'
md5_obj.update(data)  # 參數一定是字節的數據

# (3)   獲取摘要結果
ret = md5_obj.hexdigest()
print(ret)  # 828a804a719426b70b13a4e75972f8a3
~~~
